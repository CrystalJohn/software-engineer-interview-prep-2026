# Frontend Interview Prep 2025

Tài liệu chuẩn bị cho cuộc phỏng vấn Frontend 2025.

## Cấu trúc dự án

- **journal/**: Nhật ký học tập hàng ngày
- **problems/**: Bài tập và vấn đề cần giải quyết
  - arrays/: Bài tập về mảng
  - strings/: Bài tập về chuỗi
  - javascript-fundamentals/: Kiến thức cơ bản JavaScript
- **projects/**: Các dự án thực hành
- **resources/**: Tài liệu tham khảo

## Mục tiêu

Chuẩn bị toàn diện cho cuộc phỏng vấn frontend với sự tập trung vào:
- Cấu trúc dữ liệu và giải thuật
- JavaScript và Web APIs
- Frontend frameworks
- Thiết kế hệ thống
